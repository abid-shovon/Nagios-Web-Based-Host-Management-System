PART 1

Nagios Core Installation & Configuration

Ubuntu VM-e Nagios Monitoring Engine setup korar step-by-step complete documentation

Infrastructure Architecture

Nagios ecosystem-er core components gulo nicher hierarchy onujayi kaj kore:

Ubuntu VM
│
├── Apache2 (Web Server)
│
├── Nagios Core (Monitoring Engine)
│
├── Nagios Plugins (Check Tools)
│
└── Web Panel (GUI Dashboard)
Step 01

Apache Web Server Installation

Terminal Command
sudo apt install apache2 -y
Component Details & Purpose
Apache2 কি? Apache holo ekti powerful web server. Nagios GUI/Web Dashboard-ke browser-e properly render o show korar jonno Apache obossoi projon.
Step 02

Nagios Core & Plugins Installation

Terminal Command
sudo apt install nagios4 nagios-plugins nagios-plugins-contrib -y

📦 Ei command-er madhome nicher package gulo install hoy:

Package Name Function / Description
nagios4 Eti holo mool monitoring engine ja backend-e sob calculations o alerts manage kore.
nagios-plugins Host-er CPU, Disk utilization, RAM, ebong network ping check korar toolset.
nagios-plugins-contrib Community-contributed atirrikto monitoring plugins o scripts.
Step 03

Apache CGI & Rewrite Module Activation

Terminal Command
sudo a2enmod cgi rewrite
sudo systemctl restart apache2
Action Type Working Principle / Description
CGI Module Enable Nagios dynamic web interface o interactive scripts properly execute korar jonno Common Gateway Interface (CGI) support dorkar hoy.
Server Restart Apache server-ke restart korar madhome notun dynamic modules gulo active hoy.
Step 04

Nagios Authentication Password Creation

Terminal Command
sudo htpasswd -c /etc/nagios4/htpasswd.users nagiosadmin
Default User Security Details
nagiosadmin Eti Nagios dashboard access korar mool administrative user account toiri kore ebong credential file securely store rakhe.
Step 05

Services Enable & Status Verification

Terminal Command
sudo systemctl enable --now apache2
sudo systemctl enable --now nagios4
systemctl status apache2
systemctl status nagios4

💡 --now flag-ti automatic vabe service boot-time e enable korar sange sange instant start-o kore dey.

Step 06

Nagios Configuration File Verification

Terminal Command
sudo nagios4 -v /etc/nagios4/nagios.cfg

⚠️ Kono monitoring host/service create korar por configuration-e kono error ache kina ta check korar jonno eti mool command.

Step 07

UFW Firewall Configuration

Ubuntu system-er built-in firewall (UFW) active thakle nicher rule gulo allow korte hobe:

Terminal Command
sudo ufw enable
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw allow 22/tcp
sudo ufw reload
Port Number Protocol Service Target
80 TCP HTTP Unencrypted Web Access
443 TCP HTTPS Secured Web Access
22 TCP SSH Remote Terminal Access
Step 08

Accessing Nagios Web Interface

Sob kisu completed hole, je kono device-er web browser open kore nicher URL format input koro:

GET http://YOUR-SERVER-IP/nagios4

🔐 Login korar somoy Step 04 e set kora username (nagiosadmin) ebong password input korte hobe.

PART 2

Automatic Web-based Host Adding Process

Nagios Dashboard-e manual file edit chara web theke auto host add/edit/delete korar system setup

Pre-requisites: Config Check & Restart

Terminal Command
sudo nagios4 -v /etc/nagios4/nagios.cfg
sudo systemctl restart nagios4

System Workflow Diagram

http://192.168.0.239/nagios4/admin

             ▼

             │
             ▼
       [ Admin Panel ]
             │

             │
             ▼

             │

             │
             ▼

Step 01

Admin Panel Folder তৈরি করুন

Terminal Command
sudo mkdir -p /var/www/nagios-admin
sudo chown -R www-data:www-data /var/www/nagios-admin
Step 02

Apache URL Set করুন (/nagios4/admin)

File Create: /etc/apache2/conf-available/nagios4-admin.conf
sudo nano /etc/apache2/conf-available/nagios4-admin.conf

📝 Paste Inside File:

Code Snippet
Alias /nagios4/admin /var/www/nagios-admin
<Directory /var/www/nagios-admin>
    Options -Indexes
    AllowOverride AuthConfig
    Require all granted
</Directory>

⚙️ Enable & Reload:

Terminal Command
sudo a2enconf nagios4-admin
sudo systemctl reload apache2
Step 03

Password Popup Enable করুন (Basic Auth)

Terminal Command
sudo apt install apache2-utils -y
sudo htpasswd -c /etc/apache2/nagios-admin.htpasswd admin

📝 Create .htaccess file:

File Create: /var/www/nagios-admin/.htaccess
sudo nano /var/www/nagios-admin/.htaccess

📝 Paste Inside File:

Code Snippet
AuthType Basic
AuthName "Nagios Admin Panel"
AuthUserFile /etc/apache2/nagios-admin.htpasswd
Require valid-user
Step 04

Nagios Backend Manage Script তৈরি করুন

File Create: /usr/local/bin/nagios-host-manager.sh
sudo nano /usr/local/bin/nagios-host-manager.sh

📝 Paste full bash script here (You can find the shell script file in the repository).

⚙️ Grant Execute Permission:

Terminal Command
sudo chmod +x /usr/local/bin/nagios-host-manager.sh
Step 05

www-data কে Limited sudo permission দিন

Web server (www-data user) jeno securely bash script-ti chalate pare tai sudoers file-e ekti rule jog korte hobe.

Terminal Command
sudo visudo

📝 File-er ekdom sheshe giye nicher line ti paste koro:

Paste Inside File (At bottom)
www-data ALL=(ALL) NOPASSWD: /usr/local/bin/nagios-host-manager.sh

🛡️ Eta ekdom safe karon www-data user password chara shudhu matro ei ekti nirdishto script e chalate parbe, onno kono root command noy.

Step 06

Admin Panel PHP File (Frontend UI) তৈরি করুন

Eti holo mool web interface jekhan theke tumi browser er madhyome host add, edit abong delete korbe.

File Create: /var/www/nagios-admin/index.php
sudo nano /var/www/nagios-admin/index.php

📝 Nicher puro PHP code-ti copy kore file-e paste kore save koro:

Paste PHP Script inside nano

🚀 Final Execution & Verification

Sobkichu shesh hoye gele abar config file check kore Nagios reload korte hobe jate notun web panel properly kaaj kore.

Terminal Command
nagios4 -v /etc/nagios4/nagios.cfg
systemctl reload nagios4

🌐 Browser Open koro abong ei link-e jao:

GET http://SERVER-IP/nagios4/admin

🎉 Congratulations! Tumake ebar admin password prompt korbe ar login korle tumi automatic host adding dashboard-e dhuke jabe!
